# Document Templates

Frontmatter templates per category. Used by `/project-docs create`.

## Research

```yaml
---
title: "Research: Topic Name"
category: research
tags:
  - topic-tag
created: YYYY-MM-DD
updated: YYYY-MM-DD
status: draft
related:
  - "[Related Doc](../category/file.md)"
---
```

Sections: Problem Space, Options Evaluated, Comparison Matrix, Recommendation, Open Questions.

## Decision (ADR)

Filename: `NNN-slug.md` (zero-padded, sequential).

```yaml
---
title: "ADR NNN: Decision Title"
category: decision
tags:
  - area-tag
created: YYYY-MM-DD
updated: YYYY-MM-DD
status: active
related: []
---
```

Sections: Context, Decision, Consequences, Alternatives Considered.

## INDEX.md Format

Auto-generated by `/project-docs index`:

```markdown
# Documentation Index

## Architecture
- [System Name](architecture/system-name.md) — one-line summary

## Decisions
- [ADR 001: Decision](decisions/001-decision.md) — one-line summary

## Guides
- [Process Name](guides/process-name.md) — one-line summary

## Research
- [Topic Name](research/topic-name.md) — one-line summary

## API
- [Service Name](api/service-name.md) — one-line summary

---

## Archived
- [Old Doc](category/old-doc.md) — archived YYYY-MM-DD
```
